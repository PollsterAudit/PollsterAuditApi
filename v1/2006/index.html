This is a static API providing publicly available data on opinion polling in Canada.
The API is accessible at https://api.pollsteraudit.ca/

You can find the source code for this api on github

Below are the available endpoints:

  • /v1/2006/campaign_period.json
    View JSON
    Get opinion polling data between 2005-11-30 and 2006-01-22
  • /v1/2006/pre-campaign_period.json
    View JSON
    Get opinion polling data between 2005-04-07 and 2005-11-28
